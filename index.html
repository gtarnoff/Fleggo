Fleggo

Responsive Flexbox Grid System

Fleggo is my attempt at building a better grid system.

For years I have hated grids. They seem to always cause more problems than they solve. I would avoid them at all costs. Recently though I began to learn to love them, but every system I encountered had an issue in one way or another. I'm trying to solve that now.

Flexbox

Fleggo uses flexbox because it is a beutiful tool for forward thinking sites. I don't support anything older than one version below the current, so I don't worry about older versions of Internet Explorer. However, I think the way I envision Fleggo, it will work on older versions of browsers in a non-flexbox manner.

Naming Things

Most grid systems use classes like .row and .column-1, and with fleggo that's exactly what you do. But there is more.

Rows

Using flexbox actually means we don't need to define rows. The wrapping on flexbox will create them for us depeind on the width of the container. Fleggo has rows so that you can make sure all your items in a row are the same height.

Columns

Columns are at the core of any grid system. Fleggo is based on a 12 column grid and is flexible. While the initial config is based on a 960px width, columns are percentages, so you can use them regardless of the width of the container.

Since this is Flexbox, there is also a special column class, .column-0. This uses flex: 0 0 auto so that it can consume as much room as needed in a row and wrap accordingly per the flexbox spec.

Don't dirty my HTML, aka Separation of Concerns

One of my biggest complaints about grids though is the lack of semantics. I like my HTML to have classes describing the content. I find littering rows and columns in my HTML as ugly. I prefer the layout to be in the CSS and the CSS only.

This is why I built Fleggo with the ability to do just that. Yes, you can use the classes if that is your preference, but if you are like me, you can extend the row and column placeholders inside your classes for the same effect.

{% highlight sass %} .hero @extend %row .hero__title @extend %column-8 .hero__login @extend %column-4 {% endhighlight %}

This achieved the same effect of adding the .row class to the .hero block and the matching .column-* classes to the child elements.

Fleggo lets you choose the way you want to do your grid markup.
